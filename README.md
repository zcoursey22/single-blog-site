# single-blog-site
A blog made for a single admin built in Node.js.  
Project management: https://trello.com/b/99gPtm6z/single-blog-site

## Technologies
- React + React Router
- Express
- PostgreSQL hosted on AWS
- Passport.js
- Chart.js

## Features
### Admin
- Log in with email
- Change password
- Create posts
- Allow images + videos
- Edit posts
- Delete posts
- Comment on posts
- Reply to comments
- Delete any comment
- See post analytics
### User
- Search posts
- Sort posts
- Comment on posts
- Reply to comments
- Share posts to social media
- Delete comments
